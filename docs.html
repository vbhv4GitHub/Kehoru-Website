Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install kehoru-cli

NPM

$ npm install kehoru-cli

YARN 

$ yarn install kehoru-cli
